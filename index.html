Praticando com o Git e Github

O que é o GIT?

O GIT é um SCV, ou seja, sistema de controle de versões, ele controla todas as versões e alterações realizadas em um código de programação, te possibilitando acessar e modificar as mesmas.

O que é o Github

O Github é uma plataforma podendo ser acessada online ou ser baixada como Github desktop, seu objetivo é ser um rede social de códigos de versões, permitindo pessoas verem, clonarem e alterarem os códigos de outras pessoas.
